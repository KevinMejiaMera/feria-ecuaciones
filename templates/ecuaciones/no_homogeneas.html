Ecuaciones Diferenciales No Homogéneas

Métodos de Solución

Forma General

\[y'' + p(x)y' + q(x)y = g(x)\]

Donde \(g(x) \neq 0\)

Método de Coeficientes Indeterminados

Para términos no homogéneos especiales:

\[g(x) = e^{kx}, \sin(ax), \cos(ax), \text{polinomios o combinaciones}\]

Pasos:
  1. Resolver la ecuación homogénea asociada (\(y_h\))
  2. Proponer una solución particular (\(y_p\)) basada en la forma de \(g(x)\)
  3. Determinar coeficientes sustituyendo \(y_p\) en la ecuación original
  4. Solución general: \(y = y_h + y_p\)
Ejemplo:

Resolver \(y'' - 3y' + 2y = e^{3x}\):

  1. Solución homogénea: \(y_h = C_1e^x + C_2e^{2x}\)
  2. Proponer \(y_p = Ae^{3x}\)
  3. Sustituir: \(9Ae^{3x} - 9Ae^{3x} + 2Ae^{3x} = e^{3x}\) ⇒ \(A = \frac{1}{2}\)
  4. Solución: \(y = C_1e^x + C_2e^{2x} + \frac{1}{2}e^{3x}\)

Método de Variación de Parámetros

Para cualquier término no homogéneo:

\[y_p = -y_1 \int \frac{y_2 g(x)}{W(y_1,y_2)} dx + y_2 \int \frac{y_1 g(x)}{W(y_1,y_2)} dx\]

Donde \(W\) es el Wronskiano: \(W(y_1,y_2) = y_1y_2' - y_2y_1'\)

Pasos:
  1. Encontrar \(y_1\) y \(y_2\) de la solución homogénea
  2. Calcular el Wronskiano \(W(y_1,y_2)\)
  3. Calcular las integrales para obtener \(y_p\)
  4. Solución general: \(y = y_h + y_p\)
Ejemplo:

Resolver \(y'' + y = \tan x\):

  1. Solución homogénea: \(y_h = C_1\cos x + C_2\sin x\)
  2. Wronskiano: \(W = \cos x \cdot \cos x - \sin x \cdot (-\sin x) = 1\)
  3. Calcular integrales:
    • \(-\cos x \int \sin x \tan x dx = \cos x \ln|\sec x + \tan x| - \cos x \sin x\)
    • \(\sin x \int \cos x \tan x dx = -\sin x \cos x\)
  4. Solución: \(y = C_1\cos x + C_2\sin x + \cos x \ln|\sec x + \tan x| - 2\sin x \cos x\)
