Ecuaciones Diferenciales Homogéneas

Método de Solución

Forma General

\[\frac{dy}{dx} = f\left(\frac{y}{x}\right)\]

o alternativamente:

\[M(x,y)dx + N(x,y)dy = 0\]

donde M y N son funciones homogéneas del mismo grado

Paso 1: Sustitución

Realizar la sustitución \(v = \frac{y}{x}\) o \(y = vx\):

\[\frac{dy}{dx} = v + x\frac{dv}{dx}\]

Paso 2: Ecuación separable

Sustituir en la ecuación original para obtener:

\[v + x\frac{dv}{dx} = f(v)\]

que es separable:

\[\frac{dv}{f(v)-v} = \frac{dx}{x}\]

Paso 3: Integrar

Integrar ambos lados y luego sustituir de vuelta \(v = \frac{y}{x}\)

Ejemplo Resuelto

Resolver \(\frac{dy}{dx} = \frac{x^2 + y^2}{xy}\):

  1. Sustituir \(y = vx\): \(\frac{dy}{dx} = v + x\frac{dv}{dx}\)
  2. Ecuación: \(v + x\frac{dv}{dx} = \frac{1 + v^2}{v}\)
  3. Simplificar: \(x\frac{dv}{dx} = \frac{1}{v}\)
  4. Separar: \(v dv = \frac{dx}{x}\)
  5. Integrar: \(\frac{v^2}{2} = \ln|x| + C\)
  6. Solución: \(y = x\sqrt{2\ln|x| + C}\)
